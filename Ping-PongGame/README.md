# Introduction

- Created using Python and Pygame with single and double player modes available
- Player controls racket using arrow keys and catches falling balls to score


# Environment
```
OS: Windows10
Python: Python3.9
```

# Usage
```
Step1:
pip install pygame
Step2:
run "python Ping-PongGame.main.py"
```
