# Introduction
The tool for changing the background color of ID photos using the cv2 and Numpy.

# Environment
```
OS: Windows10
Python: Python3.9(have installed necessary dependencies)
```

# Usage
```
Step1:
pip install -r 
Step2:
pip install opencv-python
Step3:
run "python PhotoChange.py"
```
